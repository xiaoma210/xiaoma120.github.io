Python中的数据类型转换

Created2025-05-19|Updated2025-05-19|Python教程
|Post Views:

在 Python 中，不同的数据类型之间可以在特定场景下进行转换，例如字符串转数字、数字转字符串等。

一、常见的转换语句

  1. int(x)：将 x 转换成一个整数
  2. float(x)：将 x 转换成一个浮点数
  3. str(x)：将对象 x 转换为字符串

注意事项

  • 任意类型都可以转换为字符串；
  • 将字符串转换成数字时，必须保证字符串内容是数字，文字不可以；
  • 浮点数转整数会丢失精度（小数部分被舍弃）。

二、示例程序

# 将数字类型转换成字符串
num_str = str(11)
print(type(num_str), num_str)

# 将字符串转换成整数
num_int = int("12")
print(type(num_int), num_int)

# 将字符串转换成浮点数
num_float = float("14")
print(type(num_float), num_float)
